# Chat Bubbles for Streamer

## Overview

This project is a chat bubble application designed for streamers. It allows users to type messages that appear as chat bubbles on a web page, with customization options for bubble appearance and user profile avatars.

## Features

- **Chat Bubbles**: Displays messages as chat bubbles with customizable colors and fonts.
- **Avatar Upload**: Users can upload and set their profile avatar which appears alongside their messages.
- **Color Customization**: Customize the fill and stroke colors of the chat bubbles.
- **Typing Indicator**: Shows a typing indicator when a user is typing a message.

## Installation

To get started with the project, follow these steps:

1. **Clone the Repository**

   ```bash
   git clone https://github.com/pondsan1412/chat-bubbles-for-streamer.git
   ```

2. **Navigate to the Project Directory**

   ```bash
   cd chat-bubbles-for-streamer
   ```

3. **Install Dependencies**

   Make sure you have [Node.js](https://nodejs.org/) installed, then run:

   ```bash
   npm install
   ```

4. **Run the Development Server**

   ```bash
   npm run dev
   ```

   Open your browser and navigate to `http://localhost:5173` to view the application.

## Usage

1. **Upload Avatar**: Click on the avatar upload button to select and upload your profile picture.
2. **Type Message**: Type your message in the input field at the bottom of the screen.
3. **Submit Message**: Press `Enter` to submit your message. The chat bubble will appear with your avatar and message text.

## Customization

To customize the appearance of the chat bubbles, you can modify the CSS in the `App.css` file. You can also change the font of the text in the chat bubbles by updating the `font-family` property in `App.css`.
